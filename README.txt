# Whispread

## Description

Whispread is a web application that allows users to share posts anonymously. The application supports various types of posts including text, images, and videos. Users can view all posts without registration and the application features advanced sorting and searching functionalities.

## Features

- Anonymous posting
- Multiple post types: text, images, videos
- Advanced sorting and searching
- No registration required for viewing posts

## Tech Stack

- Front-end: React, TypeScript, Tailwind CSS
- Back-end: Express, Node.js
- Database: PostgreSQL with Sequelize

## Getting Started

### Prerequisites

- Node.js
- PostgreSQL
